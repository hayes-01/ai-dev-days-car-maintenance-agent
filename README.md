# AI Dev Days Car Maintenance Agent

A simple AI-powered vehicle diagnostic prototype for Microsoft AI Dev Days Hackathon 2026.

## Overview
This project simulates a car maintenance AI agent that analyzes vehicle data and answers user questions in natural language.  
It uses Microsoft-compatible AI tools (OpenAI/Azure OpenAI as proxy for Agent Framework) and simulated sensor data.

## Features
- Simulated real-time vehicle data (battery voltage, temperature, fault codes, etc.)
- AI agent that explains issues, gives recommendations, and reminds to consult a mechanic
- Console-based demo for quick testing

## Technologies Used
- Python
- OpenAI / Azure OpenAI API (simulating Microsoft Agent Framework)
- Simulated data generation
